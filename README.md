# yg-js-utils(函数库)

## 概述
将一些常用的前端工具函数封装，拿来即用；小伙伴们虽然公司会换，项目会换。但是工具函数不会换。来来去去就那些常用的
到底是造船过河还是借船过河，全凭个人喜好！！！作者建议不要重复造轮子，把精力放在业务逻辑上！

## 安装

npm install yg-js-utils --save
